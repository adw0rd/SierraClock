# SierraClock

Arduino clock for Ford Sierra

Requirements:

* Arduino (nano/micro)
* BME280/BMP280
* SH110X

### I²C protocol

The modules work via I²C, on Arduino it is `SDA` – `A4` pin, `SCL` – `A5` pin

### BME280/BMP280

To work with BME280/BMP280 you can use Adafruit BME280 or GyverBME280 (we used this one)
